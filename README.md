# Questoes_GIFT
Gerador de questões GIFT para aula de Projeto de Jogos

Acesse o site funcioando por [esse link](https://paulobfalmeida.github.io/Questoes_GIFT/)

O **site é estático**, ou seja ele não manda nada (então ele não envia as respostas pra mim ou pra qualquer outro lugar), suas questões são privadas pois o processamento é feito no seu computador nunca sai dele

## Criação do site
Boa parte do site foi feito com IA ([deepsite.hf.co](https://deepsite.hf.co/))

Porém eu testei e revisei bastante para ter uma boa certeza de que está funcionando corretamente
